# Klaro Capability Map

This document defines:

* what each task teaches
* how capabilities connect
* what success looks like
* how users gradually build confidence with AI

This is not a task database.

This is the behavioral learning structure behind Klaro.

---

# Core philosophy

Klaro does not aim to create AI experts.

Klaro helps users become:

* more comfortable
* more thoughtful
* more confident
* more aware

through repeated guided interaction.

Capabilities are behavioral, not technical.

---

# Capability: Asking clearly

## Learned through

* Ask a question
* Explain a screenshot

---

## User learns

Better AI responses come from clearer intent and context.

---

## Signs of success

* User gives more specific prompts
* User clarifies what they want
* User slows down before submitting

---

## Natural next capabilities

* Reviewing AI output
* Comparing responses

---

# Capability: Reviewing AI output

## Learned through

* Ask a question
* Compare products
* Fact-check information

---

## User learns

AI responses should be reviewed, not accepted automatically.

---

## Signs of success

* User compares answers
* User checks another source
* User notices uncertainty or missing context

---

## Natural next capabilities

* Fact-checking
* Decision support

---

# Capability: Protecting information

## Learned through

* Ask a question
* Write your first email
* Explain a screenshot

---

## User learns

Not all personal information should be shared with AI.

---

## Signs of success

* User removes sensitive details
* User notices private information before submitting
* User understands context matters

---

## Natural next capabilities

* Evaluating screenshots
* Sharing documents safely

---

# Capability: Breaking tasks down

## Learned through

* Plan an event or trip
* Get ideas

---

## User learns

AI can help organize thinking step-by-step.

---

## Signs of success

* User asks smaller questions
* User structures tasks more clearly
* User explores options before deciding

---

## Natural next capabilities

* Comparing options
* Decision support

---

# Capability: Comparing outputs

## Learned through

* Compare products
* Fact-check information

---

## User learns

Different prompts and sources can produce different answers.

---

## Signs of success

* User compares responses
* User notices differences in framing
* User looks beyond the first answer

---

## Natural next capabilities

* Evaluating information
* Building confidence independently

---

# Capability: Understanding visual information

## Learned through

* Explain a screenshot

---

## User learns

AI can help interpret visual content, but users should still review carefully.

---

## Signs of success

* User checks whether AI interpreted the image correctly
* User notices when context is missing
* User understands AI may miss visual nuance

---

# Capability: Using AI for everyday support

## Learned through

* Write a greeting
* Write your first email
* Plan an event or trip

---

## User learns

AI can assist with small real-world tasks without replacing personal judgment.

---

## Signs of success

* User edits AI suggestions
* User adapts responses to their own voice
* User uses AI as support rather than replacement

---

# Long-term progression philosophy

Klaro progression should feel like:

* increasing comfort
* increasing awareness
* increasing confidence

Not:

* leveling up
* mastering AI
* completing training

The system should reinforce:

> “You’re becoming more comfortable using AI thoughtfully.”

Not:

> “You are now an advanced AI user.”
